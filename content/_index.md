# Cloud Blog Challenge
A blog created with Terraform and hosted on AWS. 

Based on the [Cloud Resume Challenge](https://cloudresumechallenge.dev/docs/the-challenge/aws/) and its [Terraform Spin](https://cloudresumechallenge.dev/docs/extensions/terraform-getting-started/). 

Check out the [source code for the backend](https://github.com/sebastiankraska/cloud-resume-challenge-backend) or the [frontend](https://github.com/sebastiankraska/cloud-resume-challenge-frontend).
